################## CREATE POLICY
Select Create Policy
Modify Policy (JSON):
----------------------------
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Action": [
                "s3:*",
                "s3-object-lambda:*"
            ],
            "Resource": "*"
        }
    ]
}
----------------------------
Select Next
Select Create Policy

################## CREATE ROLE AND ASSUME ROLE FOR USER

##Create Role	
Select Create Role	
Select AWS account	
Select this account	
Select Next	
Select Policies for Role	
	Ex: AmazonS3FullAccess
Give a name to role	
Select Create Role	
	
##Assign role to user (Assume Role):	
Select User or User group	
Select Add permissions	
	Select Create inline policy:
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "Statement1",
            "Effect": "Allow",
            "Action": "sts:AssumeRole",
            "Resource": "arn:aws:iam::850740812278:role/AmazonS3FullAccessRole"
        }
    ]
}

Note: Copy ARN from role previously created
Select Next	
Give a name to policy
Select create policy
